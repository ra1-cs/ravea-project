# RAVÉA

## Beauty E-commerce Web Application

### Project Description
RAVÉA is a web-based beauty e-commerce platform that allows users to browse and purchase makeup, skincare, and beauty tools.

The platform also provides simple product recommendations based on the user's skin type and skin tone.

---

### Objectives
- Provide an easy-to-use beauty shopping platform
- Allow users to create accounts and log in
- Enable browsing and viewing products
- Allow users to manage a shopping cart and place orders
- Provide basic personalized recommendations

---

### Technologies
- Frontend: React, HTML, CSS, JavaScript
- Backend: ASP.NET Core (C#)
- Database: SQL Server

---

### Project Structure
- frontend/ → User interface
- backend/ → Server and API
- docs/ → Documentation
