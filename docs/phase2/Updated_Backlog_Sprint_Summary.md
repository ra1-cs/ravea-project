## 2. Purpose

This document summarizes the updated product backlog and sprint history for Phase II of the RAVÉA project.

The project was managed using Trello to track user stories, sprint progress, testing, documentation, and final delivery artifacts.

---

## 3. Agile Process

The project followed an Agile Scrum-inspired approach. The work was divided into two main increments:

* **Increment 1 / Sprint 1:** Core e-commerce features and first demonstrable version.
* **Increment 2 / Sprint 2:** Additional features, profile/recommendation improvements, order completion, testing, and final documentation.

This organization matches the Phase II objective of delivering the system through two working increments.
